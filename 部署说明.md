# 作品集网站部署说明

这个文件夹现在是一个纯静态作品集网站，入口文件是 `index.html`。

## 本地查看

直接双击 `index.html` 即可在浏览器打开。

## 免费发布

推荐用 Netlify Drop：

1. 打开 https://app.netlify.com/drop
2. 把整个 `作品` 文件夹拖进去
3. 等它生成网址
4. 把生成的网址发给别人

也可以用 Vercel 或 GitHub Pages。无论用哪种方式，都要上传整个文件夹，不能只上传 `index.html`，因为首页会引用 demo、文档和 `assets/previews` 里的预览图。

## 注意

`eVTOL产品研究与设计报告.pptx` 文件比较大，部分免费平台上传时可能会比较慢。如果平台提示单文件过大，可以把 PPTX 放到网盘，然后把首页里的链接改成网盘链接。
