# 智能学习助手版本改动说明

更新日期：2026-04-26

## 本次版本重点

本次更新围绕“练习更智能、计划更清晰、问答能串联练习、界面更易用”四个方向进行了较大幅度优化。

---

## 1. 学习计划模块优化

### 1.1 进度报告可视化增强
- 原“可视化进度报告”标题调整为“进度报告”。
- 删除文本复盘区域，保留更聚焦的可视化展示。
- 进度报告改成更大面积展示，不再只是小块文本信息。
- 展示内容包括：
  - 完成率
  - 任务状态分布
  - 学习节奏对比
  - 科目投入情况
  - 任务时间线

### 1.2 新建学习计划支持学习时间
- 新建学习计划时新增“学习时间”输入项。
- 学习时间会真正参与计划生成，不只是前端展示。
- 学习时间统一按 `0.5h` 为单位填写和处理。
- 即使用户输入非 0.5 的倍数，后端也会自动规范到最近的半小时粒度。

---

## 2. 练习中心能力升级

### 2.1 年级与学科体系重构
- 原“难度”维度改为“年级”维度。
- 年级范围覆盖：
  - 小学一年级到小学六年级
  - 初一到初三
  - 高一到高三
  - 大学
- 学科范围扩充，包括：
  - 语文
  - 数学
  - 英语
  - 科学
  - 物理
  - 化学
  - 生物
  - 历史
  - 地理
  - 政治
  - 高等数学
  - 大学英语
  - 计算机基础
  - 程序设计
  - Python
  - 经济学

### 2.2 题型支持增强
- 练习题型扩展为：
  - 单选
  - 多选
  - 判断
  - 简答
- 题型选择区改成更稳定的大块勾选形式。
- 支持按题型组合出题。

### 2.3 出题逻辑优化
- 修复题量选择后仍然只生成 1 道题的问题。
- 当当前题库数量不足时，系统会自动补齐同主题加练题。
- 当用户所选题型在当前学科下题量不足时，系统会自动回退到当前学科可用题型，避免直接无法生成。
- 练习中心现在支持按“知识点”精准过滤题目。

### 2.4 简答题智能判分
- 简答题不再死卡标准答案字面一致。
- 采用“关键词覆盖 + 相似度 + 表达长度”的综合判分方式。
- 对“意思差不多、关键点基本对”的答案会给出更合理的判定。

---

## 3. 错题本重构

### 3.1 显示方式优化
- 错题本由原来容易无限拉长的长列表，改成更紧凑的分页摘要模式。
- 支持展开/收起查看单题详情。
- 支持“待复习 / 已掌握”分类切换。

### 3.2 错题复习逻辑优化
- 移除“复习一次”这种手动计数式操作。
- 移除用户直接手动“标记掌握”的主流程。
- 现在必须重新作答，系统自动判题后：
  - 答对才会自动进入“已掌握”
  - 答错则继续保留在错题本
- “复习次数”改为更合理的“练对次数”逻辑。

### 3.3 错题反馈展示优化
- 待复习错题在重新作答前，不直接显示：
  - 你的答案
  - 参考答案
  - 解析
- 用户提交本次重练后，系统才显示对应反馈。
- 已掌握题目仍保留查看答案和解析能力。

### 3.4 错题数据修复
- 修复历史错题记录中的部分乱码问题。
- 修复部分错题重练中“正确答案不在选项里”的问题。
- 对单选、多选、判断题增加了选项自检与自动补正机制。

---

## 4. 本地题库扩充

### 4.1 题库数量与结构扩充
- 本地题库由原来较少的内置题，扩展为“内置题库 + 本地 CSV 题库”双层结构。
- 当前项目中已接入更大规模的本地题库内容。
- 题库覆盖从小学到大学多个年级与主要学科。

### 4.2 题库题型更完整
- 新题库支持单选、多选、判断、简答等多种题型。
- 增加知识点标签能力，为后续“问答直达练习”提供基础。

### 4.3 Python 题库补强
- 练习中心新增 Python 学科。
- 增加了 Python 基础题，如：
  - 内置数据类型
  - 函数
  - 变量
  - 列表推导式
  - 缩进规则

---

## 5. 知识问答模块升级

### 5.1 本地知识库扩充
- 增加大量基础概念知识点。
- 已补充的典型概念包括：
  - 质数
  - 常数
  - 变量
  - 分数
  - 一次函数
  - 导数
  - 主语
  - 被动语态
  - 惯性
  - 光合作用
  - Python
  - 列表推导式
  - 供求关系
  - 机会成本

### 5.2 匹配算法优化
- 不再只依赖简单关键词命中。
- 现在支持：
  - 问句中的概念抽取
  - 关键词匹配
  - 相似度兜底匹配
- 对“什么是常数”“Python 是什么”“什么是主语”这类基础问法的识别能力明显增强。

---

## 6. 知识问答与练习中心联动

### 6.1 问答结果支持推荐练习
- 问答接口不再只返回纯文本答案。
- 现在会同时返回结构化推荐练习信息，包括：
  - 推荐学科
  - 推荐年级
  - 推荐题型
  - 推荐题量
  - 对应知识点

### 6.2 一键跳转直达练习
- 用户在知识问答中提问后，如果识别到知识点：
  - 可直接点击“去练习”
  - 自动跳转到练习中心
  - 自动生成对应知识点的题目
- 例如：
  - 提问“什么是质数？”
  - 系统识别为“数学 / 小学五年级 / 判断题 / 3题 / 质数”
  - 跳转后直接出 3 道质数相关练习题

### 6.3 学习串联入口优化
- 原“去练习中心”按钮已改为“去练习”。
- “学习串联”入口和“推荐练习”入口统一为同样的智能跳转逻辑。

---

## 7. 页面文案与界面细节调整

### 7.1 界面文案优化
- 删除右上角“智能学习助手 Web 端”。
- 删除左下角“学习闭环”。
- “定制你的专注练习”改为“定制你的专属练习”。
- 删除部分冗余提示文案，让界面更简洁。

### 7.2 页面可用性优化
- 刷新页面后，新练习逻辑、错题本逻辑、问答联动逻辑都能保持正常工作。
- 前后端接口参数已经统一，不再只是页面样式变化。

---

## 8. 当前版本整体效果

与之前版本相比，本次更新后的系统更接近一个真正可用的学习辅助产品：
- 学习计划展示更直观
- 练习出题更灵活
- 简答判分更智能
- 错题复习逻辑更合理
- 知识问答不再是孤立模块
- 问答与练习已经能形成闭环
